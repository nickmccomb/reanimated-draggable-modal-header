# reanimated-draggable-modal-header

A React Native modal with a collapsible and draggable header built with Reanimated 3

## Installation

```sh
npm install reanimated-draggable-modal-header
```

## Usage


```js
import { multiply } from 'reanimated-draggable-modal-header';

// ...

const result = multiply(3, 7);
```
